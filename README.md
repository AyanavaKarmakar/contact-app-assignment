### Tech Used

- React Native
- Expo
- React Navigation
- Redux Toolkit
- React Query
- Eslint
- Prettier

### Getting Started

1. Clone the repo

   ```bash
   git clone https://github.com/AyanavaKarmakar/contact-app-assignment.git
   ```

2. Install NPM packages

   ```bash
   npm install
   ```

3. Run the app
   ```bash
   npm run start
   ```
